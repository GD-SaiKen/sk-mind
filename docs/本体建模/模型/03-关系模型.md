# 03 — 关系模型

> 定义对象类型之间如何连接。关系模型描述对象类型层面的连接规则，与业务关系图谱模型（实例层面）配套。
>
> 数据来源：`docs/本体建模/语义关系与业务规则模板.md` 第一部分 — 41条 SemanticRelation + `docs/数据梳理/ERP/脚本/generate_edges.py` — 45条边定义

## 1. 关系类型字典（8类）

| 关系大类 | 代码 | 含义 | 关联机制示例 |
| --- | --- | --- | --- |
| 结构关系 | `structural` | 组成、包含、层级、版本替代 | `BOM_D.ITEM_ID → ITEM`(父子物料) |
| 交易关系 | `transactional` | 购买、销售、供应、结算 | `SALE.CUSTOMER_BUSINESS_ID → CUSTOMER` |
| 资源关系 | `resource` | 占用、消耗、产出、调度 | `MO_MATERIAL.ITEM_ID`(投料) |
| 过程关系 | `process` | 前置、后置、触发、流转 | `SHIPPING.SOURCE_ID → SALE` |
| 责任关系 | `responsibility` | 负责、审批、执行 | — |
| 财务关系 | `financial` | 过账、核销、分配、结转 | 应收凭证→总账 |
| 质量关系 | `quality` | 检验、判定、不合格 | `PO_ARRIVAL_INSPECTION.SOURCE_ID` |
| 事件关系 | `event` | 影响、通知、触发 | — |

## 2. 关系定义模板

每个关系类型按以下模板定义：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| relation_code | string | Y | 关系编码，全局唯一 |
| relation_name | string | Y | 关系名称（英文动词） |
| relation_category | enum | Y | 关系分类 |
| subject_type | string | Y | 主体对象类型（起点） |
| object_type | string | Y | 客体对象类型（终点） |
| cardinality | string | Y | 基数约束：1:1 / 1:N / N:1 / N:M |
| join_mechanism | string | Y | 关联机制：外键 / 中间表 / 接口匹配 / 规则推导 |
| source_erp | string | N | ERP中的关联字段或表 |
| source_cross | string | N | 跨系统匹配规则 |
| trust_level | enum | Y | 可信度：high / medium |

## 5. Agent 核心查询路径

6条典型查询路径，每条标注经过的关系链：

| # | 用户问题 | Agent遍历的关系链 | 涉及关系 |
|---|---------|------------------|----------|
| Q1 | "订单 SO-001 能按时交付吗？" | SalesOrder → Shipment → Inventory(查库存) → ProductionOrder(查工单进度) → PurchaseOrder(查缺料在途) | R16, R18, R22, R13, R03 |
| Q2 | "物料 M-001 为什么库存不足？" | Material → BOM(哪些工单要用) → ProductionOrder(消耗) → PurchaseOrder(在途) → Supplier(供应能力) | R08, R12, R03, R01 |
| Q3 | "供应商 S-001 最近表现如何？" | Supplier → PurchaseOrder(交期) → PurchaseReceipt(到货) → InspectionOrder(质量) → DefectiveRecord(不良) | R02, R07, R32, R34 |
| Q4 | "工单 MO-001 的成本是多少？" | ProductionOrder → Material(投料成本) → Cost → GL | R12, R29, R30 |
| Q5 | "客户 C-001 有没有逾期应收？" | Customer → SalesOrder → AR → Payment | R16, R20, R25 |
| Q6 | "这批来料可以直接入库吗？" | PurchaseReceipt → InspectionOrder → Acceptance(检验结果) | R32, R35 |
