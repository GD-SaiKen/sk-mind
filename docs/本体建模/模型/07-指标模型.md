# 07 — 指标模型

> 定义经营指标、计算公式、分析维度、口径规则和可信等级。指标模型是 Agent 回答经营分析类问题的统一口径基础，避免「同一个指标不同人算出不同数字」。
>
> 依赖数据：ERP PURCHASE_ORDER / SALE_CARD_DOC / MO_CHANGE / ITEM_WAREHOUSE / INSPECTION；MES workorder/production 接口

## 1. 指标分类

| 层级 | 含义 | 示例 |
| --- | --- | --- |
| 基础度量 | 直接来自业务系统的原始度量值 | `ON_HAND_QTY`(库存数量)、`PLAN_QTY`(计划产量)、`AMOUNT_UNTaxed`(未税金额) |
| 业务指标 | 基于基础度量计算形成的经营指标 | 准时交付率、良率、库存周转率、产能利用率 |
| 复合指标 | 基于多个业务指标进一步组合计算 | 综合经营指数、供应商综合评分 |

## 2. 指标定义模板

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| metric_code | string | Y | 指标编码 |
| metric_name | string | Y | 指标名称 |
| metric_category | string | Y | 业务域 |
| business_definition | string | Y | 业务定义 |
| formula | string | Y | 计算公式 |
| numerator_source | string | N | 分子数据来源（表.列或接口字段） |
| denominator_source | string | N | 分母数据来源 |
| aggregation_method | enum | Y | 聚合方式 |
| time_granularity | enum | Y | 时间粒度 |
| unit | string | N | 单位 |
| available_dimensions | list | Y | 可用分析维度 |
| dependent_objects | list | Y | 依赖的BO编码 |
| dependent_attributes | list | Y | 依赖的属性编码 |
| dependent_rules | list | N | 依赖的规则编码 |
| trust_level | enum | Y | 可信等级 |

## 3. 五大核心制造指标

### 3.1 准时交付率

| 字段 | 值 |
| --- | --- |
| metric_code | `MT_ON_TIME_DELIVERY_RATE` |
| metric_name | 准时交付率 |
| metric_category | 交付 |
| business_definition | 衡量在承诺日期前完成交付的订单比例，反映企业交付履约能力 |
| formula | 准时交付订单数 / 应交付订单总数 × 100% |
| numerator_source | `SHIPPING.SHIP_DATE ≤ SALE_D.DELIVERY_DATE` 的订单数 |
| denominator_source | 应交付订单总数（排除已取消） |
| aggregation_method | 比率 |
| time_granularity | 月 |
| unit | % |
| available_dimensions | 时间、客户、产品、区域、销售域 |
| dependent_objects | `BO-SALES-ORDER`, `BO-SHIPMENT`, `BO-CUSTOMER` |
| dependent_attributes | `so_doc_no`, `doc_date`, `delivery_date`(行), `order_status` |
| dependent_rules | `FIN_AR_OVERDUE`(关联逾期) |
| trust_level | 高 |

### 3.2 生产良率

| 字段 | 值 |
| --- | --- |
| metric_code | `MT_YIELD_RATE` |
| metric_name | 生产良率 |
| metric_category | 生产 |
| business_definition | 合格产出数量占总投入数量的比例，反映生产过程质量水平 |
| formula | 合格品数量 / 总投产数量 × 100% |
| numerator_source | MES `goodQty`(报工合格数) / ERP `MO_PRODUCT.COMPLETED_QTY - SCRAP_QTY` |
| denominator_source | MES `planQty`(计划产量) / ERP `MO_CHANGE.PLAN_QTY` |
| aggregation_method | 比率 |
| time_granularity | 日 / 周 / 月 |
| unit | % |
| available_dimensions | 时间、产品、产线(WorkCenter)、班组 |
| dependent_objects | `BO-PRODUCTION-ORDER`, `BO-PRODUCT`, `BO-WORKCENTER` |
| dependent_attributes | `plan_qty`, `completed_qty`, `scrap_qty`, `good_qty`, `bad_qty` |
| dependent_rules | `MO_SCRAP_RATE_HIGH`, `MO_QUALITY_HOLD` |
| trust_level | 高 |

### 3.3 库存周转率

| 字段 | 值 |
| --- | --- |
| metric_code | `MT_INVENTORY_TURNOVER` |
| metric_name | 库存周转率 |
| metric_category | 库存 |
| business_definition | 衡量库存流转速度，反映库存管理效率 |
| formula | 期间出库量（或消耗量）/ 期间平均库存量 |
| numerator_source | 出库单据汇总（Shipment + ProductionOrder消耗） |
| denominator_source | `ITEM_WAREHOUSE.ON_HAND_QTY` 期初期末平均 |
| aggregation_method | 比率 |
| time_granularity | 月 / 季 / 年 |
| unit | 次 |
| available_dimensions | 时间、物料、仓库 |
| dependent_objects | `BO-INVENTORY`, `BO-MATERIAL`, `BO-SHIPMENT`, `BO-PRODUCTION-ORDER` |
| dependent_attributes | `on_hand_qty`, `available_qty` |
| dependent_rules | `INV_EXCESS_INVENTORY`, `INV_SAFETY_STOCK_ALERT` |
| trust_level | 中（依赖准时出库数据） |

### 3.4 产能利用率

| 字段 | 值 |
| --- | --- |
| metric_code | `MT_CAPACITY_UTILIZATION` |
| metric_name | 产能利用率 |
| metric_category | 生产 |
| business_definition | 实际产出占理论产能的比例，反映产能利用效率 |
| formula | 实际产出工时 / 可用产能工时 × 100% |
| numerator_source | 工单实际完工数量 × 标准工时 |
| denominator_source | WorkCenter 理论可用工时（班次×人数×效率） |
| aggregation_method | 比率 |
| time_granularity | 周 / 月 |
| unit | % |
| available_dimensions | 时间、产线(WorkCenter)、设备、班组 |
| dependent_objects | `BO-PRODUCTION-ORDER`, `BO-WORKCENTER`, `BO-EQUIPMENT` |
| dependent_attributes | `completed_qty`, `actual_complete_date` |
| dependent_rules | `MO_SCHEDULE_DELAY` |
| trust_level | 中 |

### 3.5 采购到货及时率

| 字段 | 值 |
| --- | --- |
| metric_code | `MT_PURCHASE_ON_TIME_RATE` |
| metric_name | 采购到货及时率 |
| metric_category | 采购 |
| business_definition | 在计划到货日期前到货的采购订单比例 |
| formula | 及时到货采购单数 / 总采购单数 × 100% |
| numerator_source | 实际到货日期 ≤ `PURCHASE_ORDER_D.PLAN_DELIVERY_DATE` 的行数 |
| denominator_source | 已审核采购订单总行数 |
| aggregation_method | 比率 |
| time_granularity | 月 |
| unit | % |
| available_dimensions | 时间、供应商、物料 |
| dependent_objects | `BO-PURCHASE-ORDER`, `BO-SUPPLIER`, `BO-MATERIAL` |
| dependent_attributes | `plan_delivery_date`(行), `status`, `close_code` |
| dependent_rules | `PO_DELIVERY_DELAY`, `PO_SUPPLIER_PERFORMANCE` |
| trust_level | 高 |

## 4. 扩展指标（P1优先级）

| 指标编码 | 指标名称 | 业务域 | 依赖BO |
| --- | --- | --- | --- |
| `MT_ORDER_FULFILLMENT` | 订单达成率 | 销售 | SalesOrder, Shipment |
| `MT_INVENTORY_DAYS` | 库存天数 | 库存 | Inventory, Cost |
| `MT_SLOW_MOVING_RATIO` | 呆滞库存占比 | 库存 | Inventory |
| `MT_FIRST_PASS_YIELD` | 一次合格率 | 质量 | Inspection, ProductionOrder |
| `MT_RETURN_RATE` | 退货率 | 质量 | ReturnOrder, SalesOrder |
| `MT_AR_TURNOVER` | 应收账款周转率 | 财务 | AR, SalesOrder |
| `MT_AP_TURNOVER` | 应付账款周转率 | 财务 | AP, PurchaseOrder |
| `MT_SUPPLIER_SCORE` | 供应商综合评分 | 采购 | Supplier, PurchaseOrder, Inspection |

## 5. 指标口径规则

| 规则编码 | 规则名称 | 说明 |
| --- | --- | --- |
| `RLMT_FILTER_CONFIRMED_ONLY` | 仅统计已确认订单 | 交付率/达成率计算中排除已取消/作废订单 |
| `RLMT_TIME_RANGE_NATURAL_MONTH` | 自然月统计周期 | 按自然月统计，非自然月需显式标注 |
| `RLMT_EXCLUDE_EXCEPTION` | 排除异常数据 | 统计时排除标记为异常的数据点 |
| `RLMT_INCLUDE_IN_TRANSIT` | 库存包含在途 | 库存周转率需标注是否含在途库存 |

Agent 回答涉及指标时，必须：1) 引用指标口径规则；2) 说明口径版本；3) 说明时间范围；4) 标注数据可信度。
