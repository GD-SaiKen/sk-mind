# 本体建模 — 原始系统数据需求清单

> 版本：v1.0  
> 日期：2026-06-30  
> 来源：基于 `docs/本体建模/` 下全部建模文档（业务对象定义、对象全景图、语义关系与规则、11个子模型）汇总提取  
> 用途：明确平台需要从 ERP/MES/SRM/TITA 四个原始系统接入的具体表、字段、API 接口

---

## 总览

| 系统 | 接入方式 | 涉及对象数 | 核心规模 |
|------|---------|-----------|---------|
| **ERP (E10)** | 数据库直连同步 | 14个BO | 26模块374表，核心14张主表 + 关联子表 |
| **MES (LightMES)** | REST API | 4个BO（物料/工单/库存/检验） | 7个核心接口 |
| **SRM (携客云)** | REST API | 7个BO（物料/供应商/客户/采购/销售/检验/发货） | 8个核心接口 |
| **TITA (CRM)** | REST API | 2个BO（客户/产品） | 4个核心接口 |

> 四系统字段模式对照：ERP用GUID主键+STRING状态码 / MES用int64+int32枚举 / SRM用String编码+int状态 / TITA用string Name标识
