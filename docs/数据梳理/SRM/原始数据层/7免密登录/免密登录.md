# 免密登录

使用场景描述

  本文档是第三方系统免密登录携客云时，用于获取动态免密登录URL的相关定义描述，供第三方相关开发人员参考。

## 1 获取免密登录URL

  此API接口是携客云生成单个用户免密登录的动态URL接口。

### 1.1 请求

#### 1.1.1 请求地址

  https://openapi.xiekeyun.com/system/ssoLogin.json

#### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

#### 1.1.3 业务请求参数（body）

| 参数名称    | 参数描述                                                     | 参数类型   | 说明                                                         | 要求     |
| ----------- | ------------------------------------------------------------ | ---------- | ------------------------------------------------------------ | -------- |
| erpCode     | 请求者用户ERP帐号                                            | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用【注】此值不是手机号对应的ERP账号，一般是绑定在一个携客云员工档案上，平台做检查使用 | **必填** |
| mobile      | 携客云用户的手机号                                           | **String** | 需要免密登录的用户对应的手机号；【注】此手机号需要已经在携客云平台注册了账号 | **必填** |
| mobileArea  | 手机区域码                                                   | **String** | 国内: +86 香港: +852 台湾: +886 澳门: +853 韩国: +82 越南: +84 新加坡: +65 日本: +81 泰国: +66 美国&加拿大: +1 | **必填** |
| isConstrain | 是否限制跳转到指定企业 0:不限；1：限制，默认：0 ，针对一个用户存在多企业场景使用 | **Number** | 此参数控制获取到的ssoUrl,根据此URL跳转到携客云时，是否强制跳转到请求时传入的企业中; 如果不传值或传值=0，免密登录到携客云后，跳转到用户的默认企业；如果传值=1，返回ssoUrl中会返回参数oauth_callback | 选填     |

#### 1.1.4 请求参数示例

```
{
  "commonParam":{
    "appKey":"3da512e5eb0aa2fa27b6c3ffc638da22",
    "version":"1.0",
    "ownerCompanyCode":"91904378",
    "operateCompanyCode":"91904378",
    "sign":"de95ea0786a7b5f52a5ec728aca91086",
    "timestamps":1620960936
  },
  "body":{
    "erpCode":"00001",
    "mobile":"1881234678",
    "mobileArea":"+86"
  }
}
    
```

#### 1.1.5 说明

  同一个携客云用户手机号，访问频率不能低于 **15** 秒；

### 1.2 结果返回

#### 1.2.1 描述

  结果数据以Json格式返回

#### 1.2.2 返回结果

##### 1.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| **data**  | 接口成功时(result=1)返回                                     | **Json对象** |

##### 1.2.2.2 JSON对象(data)字段定义

| 字段名称     | 描述                                                       | 类型       |
| ------------ | ---------------------------------------------------------- | ---------- |
| ssoUrl       | 免密登录URL， **此URL有效期30秒**                          | **String** |
| employeeName | 员工姓名; 用户在当前企业(ownerCompanyCode)中对应的员工姓名 | **String** |

##### 1.2.2.3 返回示例

###### 1.2.2.3.1 请求成功例子

```
   {
  "result":1,
  "data":{
    "ssoUrl":"[https://console.xiekeyun.com/login/xky/callback?ssoToken=ecd4dd73442b7d56f6e2f503713dda6a&checkToken=219d83630d116f1e5cbdf8db1ecb030d&ts=1620961793622](https://console.xiekeyun.com/login/xky/callback?ssoToken=ecd4dd73442b7d56f6e2f503713dda6a&checkToken=219d83630d116f1e5cbdf8db1ecb030d&ts=1620961793622)",
    "employeeName":"张三"
  }
}
    
```

###### 1.2.2.3.2 请求失败例子

{
  "errorCode":"",
  "errorMsg":"携客云平台不存在此手机号对应的用户",
  "result":"9",
