# 附件上传下载

场景描述

此API实现客户把附件上传至携客云或从携客云拉取单据的附件相关操作的API接口文档，供第三方相关开发人员参考。

其中包括：

- [附件下载](https://openapi.xiekeyun.com/doc/docfile/download.html#附件下载)

- [附件上传附件下载

  ​     此API接口是第三方系统(ERP)根据自己需求，把企业在携客云平台单据中的附件下载到内部。

  ​     **把一个单据中的附件下载到本地，开发者需要以下两个大步骤：**

    **步骤1：**获取某一单据中各附件的下载地址

    **步骤2：**下载文件流（二进制字节流）

## 2 上传文件流

  通过已获取到的各个 uploadUrl（文件上传地址）使用 POST 请求方法把文件流（二进制字节流）上传到服务端。

### 2.1 请求

#### 2.1.1 请求地址

  【[获取某一单据中各附件的上传地址](https://openapi.xiekeyun.com/doc/docfile/upload.html#1-获取某一单据中各附件的上传地址)】接口中返回的各个uploadUrl

#### 2.1.2 请求方式

  **POST**

#### 2.1.3 Content-Type

  **multipart/form-data**

#### 2.1.4 数据提交方式

  **form**

#### 2.1.5 文件流提交对应的字段名称

  **file**

#### 2.1.6 请求示例（postman调用）

![img](https://openapi.xiekeyun.com/doc/images/upload_postman.png)

#### 2.1.7 代码请求参数示例

Demo

```java
package com.test.service;


import org.apache.http.HttpEntity;
import org.apache.http.HttpResponse;
import org.apache.http.client.HttpClient;
import org.apache.http.client.methods.HttpPost;
import org.apache.http.entity.ContentType;
import org.apache.http.entity.mime.HttpMultipartMode;
import org.apache.http.entity.mime.MultipartEntityBuilder;
import org.apache.http.impl.client.HttpClients;
import org.apache.http.util.EntityUtils;

import java.io.File;
import java.nio.charset.StandardCharsets;

public class FileUploadExample {
    public static void main(String[] args) {
        try {
            // 本地文件路径
            String filePath = "D:\\Downloads\\3.png";
            // 目标上传URL
            String targetUrl = "https://fileapi.xiekeyun.com/upload/crVGAbvwD7rmZoiX66HKazjj4dl7XlXB";

            // 创建HttpClient实例
            HttpClient httpClient = HttpClients.createDefault();

            // 创建HttpPost实例
            HttpPost httpPost = new HttpPost(targetUrl);

            // 设置文件上传
            File file = new File(filePath);
            MultipartEntityBuilder builder = MultipartEntityBuilder.create();
            builder.setCharset(StandardCharsets.UTF_8);
            builder.setMode(HttpMultipartMode.BROWSER_COMPATIBLE);
            /*
             *【注】
             *      第3个参数需要传入上传文件对应的ContentType，此值是根据用户实际上传文件对应的具体类型值
             *      可以参考org.apache.http.entity.ContentType.java中定义的各类型
             */
            builder.addBinaryBody("file", file, XXXXXXX, file.getName());

            // 构建HttpEntity
            HttpEntity multipart = builder.build();

            // 设置请求实体
            httpPost.setEntity(multipart);

            // 执行上传
            HttpResponse response = httpClient.execute(httpPost);

            // 打印响应状态
            System.out.println("Response Code : " + response.getStatusLine().getStatusCode());

            // 打印响应内容
            System.out.println(EntityUtils.toString(response.getEntity()));

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

### 4.1 请求

#### 4.1.1 请求地址

#### 4.1.2 请求方式

  POST

#### 4.1.3 Content-Type

  application/json; charset=utf-8

#### 4.1.4 请求公传参数（commonParam）

#### 4.1.5 业务请求参数（body）
