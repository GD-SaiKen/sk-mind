## 1 对账单列表查询

  此查询API接口是查找对账单列表信息

## 1.1 请求

### 1.1.1 请求地址

https://openapi.xiekeyun.com/customer/statement/getList.json

### 1.1.2 请求公传参数（commonParam）

 **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】 所述**

### 1.1.3 业务请求参数（body）

| 参数名称  | 参数描述                       | 参数类型       | 说明                                                         | 要求     |
| --------- | ------------------------------ | -------------- | ------------------------------------------------------------ | -------- |
| startDate | 时间范围的开始时间（发布时间） | **Number**     | 开始时间对应的时间戳数值（毫秒）; 查询时为闭区间（即>=startDate） | **必填** |
| endDate   | 时间范围的结束时间（发布时间） | **Number**     | 结束时间对应的时间戳数值（毫秒）; 查询时为闭区间（即 <=endDate）(startDate,endDate]最大时间区间为12小时 | **必填** |
| erpCode   | 请求者用户ERP帐号              | **String**     | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，返回此员工有权限的数据 | **必填** |
| status    | 单据状态值                     | **Number数组** | 对账单状态说明（1:待采购方确认,5:已确认,6:数量待确认（开启数量确认的时候，才有这个）,7:待供应方确认（采购方发起对账，并要求供应商确认）） | **选填** |

    "appKey":"2505309c9757c3246ed317775a861441",

    "version":"1.0",

  },

      5

    ],

  }

### 1.1.5 说明

  \1. 本接口访问频率不能低于 **5** 分钟；

  \2. 一次请求中开始结束时间范围不能大于 **12** 小时；

### 1.2.1 描述

### 1.2.2 返回结果

| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |

| innerCustomerName | 内部客户名称（ERP）                                          | **String** |
