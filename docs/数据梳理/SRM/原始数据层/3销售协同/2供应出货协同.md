## 1 送货单号列表查询

此查询API接口是仓库系统按时间范围拉取新的送货单号列表功能

## 1.1 请求

### 1.1.1 请求地址

https://openapi.xiekeyun.com/customer/delivery/getNoList.json

### 1.1.2 请求公传参数（commonParam）

 **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

  **备注：**当版本号 **"version"**:**"1.0"** 时，：

​      开始时间和结束时间对应的是送货单发布时间

​       当版本号 **"version"**:**"2.0"** 时，查询出状态是：

​      开始时间和结束时间对应的是送货单异动时间       **以下操作会更新送货单异动时间**

​      1）供应商更新送货方式相关信息

​      2）供应商更新条码信息

​      3）供应商申请送货

​      4）供应商确认发出

​      5）供应商撤回送货

​      6）供应商货物已发

​      7）采购方更新条码信息

​      8）采购方货物到达

​      9）采购方签收相关操作

​      10）采购方确认收货

​      11）采购方退回

​      12）采购方退回申请

### 1.1.3 业务请求参数（body）

| 参数名称          | 参数描述          | 参数类型                  | 说明                                                         | 要求     |
| ----------------- | ----------------- | ------------------------- | ------------------------------------------------------------ | -------- |
| startDate         | 开始时间          | **Number**                | 开始时间对应的**毫秒**时间戳数值; 查询时为开区间（即 >startDate） | **必填** |
| endDate           | 结束时间          | **Number**                | 结束时间对应的**毫秒**时间戳数值; 查询时为闭区间（即 <=endDate） | **必填** |
| erpCode           | 请求者用户ERP帐号 | **String**                | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，返回此员工有权限的数据 | **必填** |
| innerCustomerCode | 内部客户编码      | **String**                | 客户档案中的客户编码，如果传此值，只返回此客户相关的数据     | **选填** |
| status            | 状态              | **number数组，例：[1,3]** | 不传时，返回所有状态[1,3,5,6,7,9]，依次对应【待签收, 已完成, 申请中, 退回, 已同意, 部分同意】, 传值时只支持这些状态 | **选填** |

### 1.1.5 说明

  \1. 本接口访问频率不能低于 **5** 分钟；

  \2. 一次请求中时间范围不能大于 **24** 小时；

## 1.2 结果返回

### 1.2.1 描述

结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

| --------- | ------------------------------------------------------------ | ------------ |

| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 送货单号数据列表                                             | **JSON数组** |

### 1.2.2.2 dataList中一个Json数据明细字段定义

| 字段名称          | 描述                                                         | 类型       |
| ----------------- | ------------------------------------------------------------ | ---------- |

| status            | 1.待签收, 3.已完成, 5.申请中, 6.退回, 7.已同意, 9.部分同意   | **String** |
