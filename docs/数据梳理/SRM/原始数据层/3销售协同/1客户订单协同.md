## 1 客户订单列表查询

  此API接口是提供给销售方获取客户订单列表数据。

## 1.1 请求

### 1.1.1 请求地址

  https://openapi.xiekeyun.com/customerOrder/orderList.json

### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称             | 参数描述                                                     | 参数类型   | 说明                                                         | 要求     |
| -------------------- | ------------------------------------------------------------ | ---------- | ------------------------------------------------------------ | -------- |
| orderStatus          | 订单状态 1:待答交 2:差异待确认 3:退回待答交 4:变更确认中 5:订单已确认 6:已结案 7:已冻结 8:已留置 10:撤回答交 11:采购方撤回 12:作废 13:供应商方拒绝 | **Number** | 查询时，如果传值，只返回此状态对应的订单                     | **选填** |
| publishStartTime     | 发布开始时间（订单发布时间）                                 | **Number** | 发布时间与确认时间，至少有一对必须有值；开始时间对应的**毫秒**时间戳数值; 查询时为闭区间（即 >= publishStartTime）最大起止范围24小时 | **必填** |
| publishEndTime       | 发布结束时间（订单发布时间）                                 | **Number** | 发布时间与确认时间，至少有一对必须有值；结束时间对应的**毫秒**时间戳数值; 查询时为闭区间（即 <= publishEndTime）最大起止范围24小时 | **必填** |
| confirmStartTime     | 确认时间起始（答交确认时间）                                 | **Number** | 发布时间与确认时间，至少有一对必须有值；开始时间对应的**毫秒**时间戳数值; 查询时为闭区间（即 >= confirmStartTime）最大起止范围24小时 | **必填** |
| confirmEndTime       | 确认时间截止（答交确认时间）                                 | **Number** | 发布时间与确认时间，至少有一对必须有值；结束时间对应的**毫秒**时间戳数值; 查询时为闭区间（即 <= confirmEndTime）最大起止范围24小时 | **必填** |
| erpCode              | 请求者用户ERP帐号                                            | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，用于判断此员工在平台是否有此数据权限 | **必填** |
| customerXkUniqueCode | 客户档案平台唯一编码                                         | **String** |                                                              | **选填** |
| innerCustomerCode    | 内部客户编码                                                 | **String** |                                                              | **选填** |

## 2 客户订单详情数据查询

  此查询API接口是第三方系统(ERP)按平台单据号获取详情数据。

## 2.1 请求

### 2.1.1 请求地址

  https://openapi.xiekeyun.com/customerOrder/orderDetail.json

### 2.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 2.1.3 业务请求参数（body）

| 参数名称         | 参数描述               | 参数类型   | 说明                                                         | 要求     |
| ---------------- | ---------------------- | ---------- | ------------------------------------------------------------ | -------- |
| poXkNo           | 平台单据号             | **String** |                                                              | **必填** |
| erpCode          | 请求者用户ERP帐号      | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，用于判断此员工在平台是否有此数据权限 | **必填** |
| returnAnswerFlag | 是否返回订单的答交信息 | **Number** | 是否返回订单的答交信息 0：不返回；1:返回                     | **选填** |

### 2.1.4 请求参数示例

Demo

```
{

  "commonParam":{

    "appKey":"2505309c9757c3246ed317775a861441",

    "version":"1.0",

    "ownerCompanyCode":"94563778",

    "operateCompanyCode":"94563778",

"sign": "74fc0a7faa0933b41b6ea06856b2518f",

"timestamps": "1606918844"

  },

"body":{

         "poXkNo":"1017733110305727",

        "erpCode":"000001",

    "returnAnswerFlag":1

  }

}
    
```

### 2.1.5 说明

  本接口查询同一个订单的答交数据时，访问频率不能低于 **6** 秒钟；
