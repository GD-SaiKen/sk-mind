## 1 客户物料询价招标列表查询

  此API接口是提供给销售方获取客户询价招标列表信息

## 1.1 请求

### 1.1.1 请求地址

  https://openapi.xiekeyun.com/customer/inquiryOrder/getList.json

### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

  **备注：**当版本号 "version":"1.0" 时，按询价单发布时间获取列表；         当版本号 "version":"2.0" 时，按询价单更新时间获取列表；

### 1.1.3 业务请求参数（body）

| 参数名称  | 参数描述               | 参数类型   | 说明                                                         | 要求     |
| --------- | ---------------------- | ---------- | ------------------------------------------------------------ | -------- |
| startDate | 客户询价单发布开始时间 | **Number** | 开始时间对应的**毫秒**时间戳数值; 查询时为闭区间             | **必填** |
| endDate   | 客户询价单发布结束时间 | **Number** | 开始时间对应的**毫秒**时间戳数值; 查询时为闭区间             | **必填** |
| erpCode   | 请求者用户ERP帐号      | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，用于判断此员工在平台是否有此数据权限 | **必填** |

### 1.1.4 请求参数示例

Demo

```
{

  "commonParam": {

    "appKey": "2505309c9757c3246ed317775a861441",

    "version": "1.0",

    "ownerCompanyCode": "94563778",

    "operateCompanyCode": "94563778",

    "sign": "1e48ac2ea0d3aae7ec2fe4e0f0d4f4c4",

    "timestamps": "1640326132"

  },

"body": {

        "startDate": 1640355129000,

"endDate": 1640355129000,

    "erpCode": "000001"

  }

}
    
```

### 1.1.5 说明

  本查询接口访问频率不能低于 **5** 分钟；

## 1.2 结果返回

### 1.2.1 描述

  结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型             |
| --------- | ------------------------------------------------------------ | ---------------- |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**       |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**       |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**       |
| dataList  | 订单号列表数据                                               | **Json对象数组** |

### 1.2.2.2 JSON对象(dataList)字段定义说明

| 字段名称 | 描述                                                     | 类型       |
| -------- | -------------------------------------------------------- | ---------- |
| inqXkNo  | 询价单号                                                 | **String** |
| status   | 单据状态 1未开始 2报价中 3已取消 4比价中 5已完成 6已结束 | **Number** |

### 1.2.3 查询返回示例

Demo

```
{

    {
      "inqXkNo": "502112201399",
      "status": 1
    }
  ]
}
    
```

## 1 客户物料询价招标报价列表查询

## 1.1 请求

  https://openapi.xiekeyun.com/customer/inquiryOrder/getQuoteList.json

### 1.1.2 请求公传参数（commonParam）

| 参数名称          | 参数描述               | 参数类型   | 说明                                                         | 要求     |
