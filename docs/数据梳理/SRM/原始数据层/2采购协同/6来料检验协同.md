## 1 检验单列表查询

  此查询API接口是查询检验单列表信息。

## 1.1 请求

### 1.1.1 请求地址

  https://openapi.xiekeyun.com/verifyOrder/getList.json

### 1.1.2请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称  | 参数描述          | 参数类型      | 说明                                                         | 要求     |
| --------- | ----------------- | ------------- | ------------------------------------------------------------ | -------- |
| startDate | 更新时间          | **Number**    | 开始时间对应的时间戳数值; 查询时为开区间（即 >startDate）    | **必填** |
| endDate   | 更新时间          | **Number**    | 结束时间对应的时间戳数值; 查询时为闭区间（即 <=endDate）(startDate,endDate]最大时间区间为120分钟 | **必填** |
| erpCode   | 请求者用户ERP帐号 | **String**    | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，返回此员工有权限的数据 | **必填** |
| status    | 来料检验单状态    | **Integer[]** | status状态说明：0-检验中, 1-待确认,2-退回,3-已确认,4-已作废,5-待发出,6-撤回；该字段没有传值时，默认返回“已确认“状态的单据 | **选填** |

### 1.1.4 请求参数示例

Demo

```
{

  "commonParam":{

    "appKey":"3da512e5eb0aa2fa27b6c3ffc638da22",

    "version":"1.0",

    "ownerCompanyCode":"91904378",

    "operateCompanyCode":"91904378",

    "sign":"c257fa4b6568936f2309f0629c557dd0",

    "timestamps":"1568802058"

 },

  "body":{

    "erpCode":"liuzy",

    "startDate":1568717105000,

    "endDate":1568721600000

  }

}
    
```

### 1.1.5 说明

  \1. 本接口访问频率不能低于 **18** 秒；

  \2. 一次请求中开始结束时间范围不能大于 **2** 小时；

  \3. 查询数据时，不能查询 **30** 天之前的数据。

### 1.2.1 描述

  结果数据以Json格式返回

| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 检验单列表                                                   | **Json数组** |
