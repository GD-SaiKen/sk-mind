## 4 查询供应商应用度报表

此API接口是查询携客云平台中，查询供应商应用报表数据

### 4.1 请求

#### 4.1.1 请求地址

https://openapi.xiekeyun.com/compVendor/report.json

#### 4.1.2 请求公传参数（commonParam）

**请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】 所述**

#### 4.1.3 业务请求参数（body）

| 参数名称 | 参数描述          | 参数类型   | 说明                                                         | 要求     |
| -------- | ----------------- | ---------- | ------------------------------------------------------------ | -------- |
| erpCode  | 请求者用户ERP帐号 | **Number** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据 | **必填** |
| year     | 年份              | **Number** |                                                              | **必填** |
| month    | 月份              | **Number** |                                                              | **必填** |
| type     | 查询类型          | **Number** | 查询类型: 1:公司 2:组织 默认1; **(如果为2时，当前企业开启采购组织，返回当前企业用户及关联企业有权限的企业组织信息；未开启采购组织时，直接返回空)** | **选填** |
