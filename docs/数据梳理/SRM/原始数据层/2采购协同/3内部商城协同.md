## 1 内部商城引入单列表查询

  此API接口是查询内部商城引入单列表数据

## 1.1 请求

### 1.1.1 请求地址

  https://openapi.xiekeyun.com/introduceOrder/getList.json

### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称  | 参数描述           | 参数类型   | 说明                                                         | 要求     |
| --------- | ------------------ | ---------- | ------------------------------------------------------------ | -------- |
| erpCode   | 请求者用户ERP帐号  | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据 | **必填** |
| startDate | 引入时间的开始时间 | **Number** | 引入时间的时间对应的时间戳数值;查询时为开区间（即 >= updateTime） | **必填** |
| endDate   | 引入时间的结束时间 | **Number** | 引入时间的时间对应的时间戳数值;查询时为开区间（即 <= updateTime）(startDate,endDate]最大时间区间为一天 | **必填** |

### 1.1.4 请求参数示例

Demo

```
{
    "commonParam": {
        "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",
        "version": "1.0",
        "ownerCompanyCode": "91904378",
        "operateCompanyCode": "91904378",
        "sign": "89f35004a8262c89040da64e627ed981",
        "timestamps": 1670907414
    },
    "body": {
        "erpCode": "account24055",
        "startDate": 1668052553000,
        "endDate": 1671508553000
    }
}
    
```

### 1.1.5 说明

  \1. 本接口访问频率不能低于 **5** 分钟；

## 1.2 结果返回

### 1.2.1 描述

  结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.3 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 内部商城引入单列表                                           | **JSON数组** |

### 1.2.4 返回结果dataList公共字段定义

| 字段名称   | 描述                                                         | 类型       |
| ---------- | ------------------------------------------------------------ | ---------- |
| inductXkNo | 内部商城引入单平台单据号                                     | **String** |
| status     | 内部商城引入单状态 0:草稿 1:待选品 2:待发布 3:已完成 4:已撤回 5:已作废 | **Number** |
