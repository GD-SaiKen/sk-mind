## 1 请购单列表查询

  此查询API接口是查找请购单列表信息

## 1.1 请求

### 1.1.1 请求地址

  https://openapi.xiekeyun.com/requisition/getList.json

### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称  | 参数描述          | 参数类型   | 说明                                                         | 要求     |
| --------- | ----------------- | ---------- | ------------------------------------------------------------ | -------- |
| startDate | 请购时间          | **Number** | 开始时间对应的时间戳数值; 查询时为闭区间（即 >=startDate）。**起止时间不支持查询30天之前数据** | **必填** |
| endDate   | 请购时间          | **Number** | 结束时间对应的时间戳数值; 查询时为闭区间（即 <=endDate）。如果status传值，(startDate,endDate]最大时间区间为5天；否则最大时间区间为1天 | **必填** |
| dateType  | 请购时间          | **Number** | 查询时间类型，1：请购时间，2：更新时间，3：发布时间，为空默认查询 请购时间 | **必填** |
| erpCode   | 请求者用户ERP帐号 | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，返回此员工有权限的数据 | **必填** |
| status    | 请购单的状态      | **Number** | 请求时，如果传入此值，只返回传入状态的单号列表，1: 待转采购；2: 部分转采购；3：已完成;4：冻结；5：结案；6：部分转询价；7：已转询价；8：已撤回；9：已作废;10：待转询价;11：已转竞价;12：取消 | **选填** |

### 1.1.4 请求参数示例

Demo

```
{
  "commonParam":{
    "appKey":"3da512e5eb0aa2fa27b6c3ffc638da22",
    "version":"1.0",
    "ownerCompanyCode":"91904378",
    "operateCompanyCode":"91904378",
    "sign":"75faa9d5794474fb1f8ba84c892bee4b",
    "timestamps":"1563948991"
  },
  "body":{
    "erpCode":"testhe",
    "startDate":1563811200000,
    "endDate":1563814800000,
    "dateType": 1
  }
}
    
```

### 1.1.5 说明

  \1. 本接口访问频率不能低于 **30** 秒；

  \2. 一次请求中开始结束时间范围：不能大于 **1** 天

  \3. 查询数据时，不能查询 **30** 天之前的数据。

## 1.2 结果返回

### 1.2.1 描述

  结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 请购单号数据列表                                             | **JSON数组** |

### 1.2.2.2 dataList字段定义

| 字段名称   | 描述                                     | 类型       |
| ---------- | ---------------------------------------- | ---------- |
| reqXkNo    | 请购单携客云平台单号                     | **String** |
| reqInnerNo | 请购单ERP单据号                          | **String** |
| status     | 单据状态：1:待审核；2:已完成；3：已作废; | **Number** |

## 1 已答交的采购单号列表查询

  此API接口是提供给第三方系统(ERP)，按答交时间范围查询采购单号列表。

## 1.1 请求

### 1.1.1 请求地址

  https://openapi.xiekeyun.com/purchase/list.json

### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称         | 参数描述                                                     | 参数类型   | 说明                                                         | 要求     |
| ---------------- | ------------------------------------------------------------ | ---------- | ------------------------------------------------------------ | -------- |
| answerStartDate  | 采购单答交的开始时间                                         | **Number** | 开始时间对应的**毫秒**时间戳数值; 查询时为闭区间（即 >=answerStartTime） | **必填** |
| answerEndDate    | 采购单答交的结束时间                                         | **Number** | 结束时间对应的**毫秒**时间戳数值; 查询时为闭区间（即 <=answerEndTime） | **必填** |
| erpCode          | 请求者用户ERP帐号                                            | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，用于判断此员工在平台是否有此数据权限 | **必填** |
| innerVendorCode  | 内部供应商编码                                               | **String** | 供应商档案中的供应商编码，如果传值，只查询此供应商的数据     | **选填** |
| orderStatus      | 采购单状态定义（2:交期差异待确认, 3:退回待供应商答交,4:变更确认中,5:订单已确认, 6:已结案, 7:已冻结, 8:已留置,10:撤回答交,13:供应商方拒绝） | **Number** | 查询时，如果传值，只返回此状态对应的已答交订单；             | **选填** |
| diffFlag         | 答交是否有差异 0：无差异；1：有差异                          | **Number** | 此值表示供应商答交的采购单是否有差异（单价、数量、交期）；如果传值只返回供应商答交有差异的单据 | **选填** |
| profitCenterCode | 收货部门编码                                                 | **String** | 对应订单单身中的收货部门编码                                 | **选填** |
