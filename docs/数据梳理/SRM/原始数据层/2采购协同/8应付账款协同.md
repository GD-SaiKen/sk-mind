## 1 对账单列表查询

  此查询API接口是查找对账单列表信息

## 1.1 请求

### 1.1.1 请求地址

  https://openapi.xiekeyun.com/vStatementOrder/getList.json

### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称        | 参数描述             | 参数类型   | 说明                                                         | 要求     |
| --------------- | -------------------- | ---------- | ------------------------------------------------------------ | -------- |
| updateTimeStart | 最后编辑时间，从     | **Number** | 开始时间对应的时间戳数值（毫秒）; 查询时为闭区间（即>=updateTimeStart） | **必填** |
| updateTimeEnd   | 最后编辑时间，到     | **Number** | 结束时间对应的时间戳数值; 查询时为闭区间（即 <=updateTimeEnd）(updateTimeStart,updateTimeEnd]最大时间区间为12小时 | **必填** |
| erpCode         | 请求者用户ERP帐号    | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，返回此员工有权限的数据 | **必填** |
| status          | 单据状态值           | **Number** | 0:草稿 1:待采购方确认 2:撤回 3:作废 4:退回 5:已确认 6:数量待确认 7:待供应商确认 | **选填** |
| syncStatus      | 同步状态值           | **Number** | null、0:未同步 1:同步中 2:同步成功 3:同步失败                | **选填** |
| esignStatus     | 签章状态值           | **Number** | null、0:未签章 1:签章中 2:签章完成                           | **选填** |
| submitCheckFlag | 过滤只提交送审的数据 | **Number** | 为1时只查询已提交送审的数据                                  | **选填** |

### 1.1.4 请求参数示例

Demo

```
{
  "commonParam":{
    "appKey":"3da512e5eb0aa2fa27b6c3ffc638da22",
    "version":"1.0",
    "ownerCompanyCode":"91904378",
    "operateCompanyCode":"91904378",
    "sign":"75faa9d5794474fb1f8ba84c892bee4b",
    "timestamps":"1563948991"
  },
  "body":{
    "erpCode":"XK699",
     "status":3,
    "updateTimeStart":1571767564000,
    "updateTimeEnd":1571796364000
    }
}
    
```

### 1.1.5 说明

  \1. 本接口访问频率不能低于 **5** 分钟；

  \2. 一次请求中开始结束时间范围不能大于 **30** 天；

## 1.2 结果返回

### 1.2.1 描述

  结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 对账单单号数据列表                                           | **JSON数组** |

### 1.2.2.2 dataList中一个Json数据明细字段定义

| 字段名称            | 描述                                                         | 类型       |
| ------------------- | ------------------------------------------------------------ | ---------- |
| recoXkNo            | 对对账单携客云平台单号                                       | **String** |

| syncStatus          | null、0:未同步 1:同步中 2:同步成功 3:同步失败                | **Number** |
| innerVendorCode     | 内部供应商编码，对erp编码                                    | **String** |
| ~~checkStatusCode~~ | ~~流程状态编码~~                                             | **String** |
| ~~checkStatusName~~ | ~~流程状态名称~~                                             | **String** |
| flowStatus          | 流程当前审批节点状态(0:不同意 1:部分同意 2：全部同意)        | **Number** |
| endFlowFlag         | 流程是否结束状态；0：未终止 1：终止同意 2：终止拒绝          | **Number** |
