# 附件更新查询

概述

**修订历史记录**

| **日期**   | **版本** | **说明** | **作者** |
| ---------- | -------- | -------- | -------- |
| 2025/06/20 | V1.0.0   | 新增接口 | lvsd     |


  此API接口用于查询相应业务单据编码，场景：供应商有调整上传的附件时，调用此接口可以获取对应的单据编码，包括1个API接口：

###   [获取供应商新上传了附件的单据-列表](https://openapi.xiekeyun.com/docpull/order/orderhasuploadnewfile.md)

​     **调用场景：**

​       采购企业想知道针对某一类型的单据在某个时间段内，供应商是否有操作附件（上传/删除）， 通过此API接口可以查询对应单据编码列表数据；

## 1 获取供应商新上传了附件的单据列表

  通过此API接口，查询某一类型的单据在**(**startDate,endDate**]**时间范围内，供应商有操作附件（上传/删除）的单据编码列表。

### 1.1 请求

#### 1.1.1 请求地址

  https://openapi.xiekeyun.com/orderHasUploadFile/queryByLastDate.json

#### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

#### 1.1.3 业务请求参数（body）

| 参数名称   | 参数描述           | 参数类型   | 说明                                                         | 要求     |
| ---------- | ------------------ | ---------- | ------------------------------------------------------------ | -------- |
| erpCode    | 请求者用户ERP帐号  | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，返回此员工有权限的数据 | **必填** |
| startDate  | 时间范围的开始时间 | **Number** | 开始时间对应的时间戳数值; 查询时为开区间（即 >startDate）    | **必填** |
| endDate    | 时间范围的结束时间 | **Number** | 结束时间对应的时间戳数值; 查询时为闭区间（即 <=endDate）(startDate,endDate]最大时间区间为2小时 | **必填** |
| billType   | 单据类型           | **Number** | 10:采购订单                                                  | **必填** |
| uploadType | 附件上传者类型     | **Number** | 1：采购方上传； 2：供应商方上传 （目前只支持**2**） 默认=2   | 选填     |

### 1.2 结果返回

#### 1.2.1 描述

  结果数据以Json格式返回

#### 1.2.2 返回结果

#### 1.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 单号数据列表                                                 | **JSON数组** |

#### 1.2.2.2 dataList中一个Json数据明细字段定义

| 字段名称  | 描述         | 类型       |
| --------- | ------------ | ---------- |
| billXkNo  | 单据平台编码 | **String** |
| billErpNo | 单据ERP编码  | **String** |
