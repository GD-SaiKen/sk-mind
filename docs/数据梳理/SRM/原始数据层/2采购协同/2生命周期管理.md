### 1 潜在供应商列表查询

此查询API接口是查询一个企业潜在供应商列表数据的服务。

#### 1.1 请求

##### 1.1.1 请求地址

https://openapi.xiekeyun.com/potentialVendor/getList.json

##### 1.1.2 请求公传参数（commonParam）

**请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

##### 1.1.3 业务请求参数（body）

| 参数名称            | 参数描述            | 参数类型   | 说明                                                         | 要求 |
| ------------------- | ------------------- | ---------- | ------------------------------------------------------------ | ---- |
| erpCode             | 请求者用户ERP帐号   | String     | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用 | 必填 |
| dateStart           | 资质审核完成时间-从 | Number     | 开始时间对应的 毫秒 时间戳数值; 查询间隔最大不超过24小时，资质审核时间范围和创建时间、审核时间、转正时间范围必填一项 结束时间对应的 毫秒 时间戳数值; | 必填 |
| dateEnd             | 资质审核完成时间-至 |            |                                                              |      |
| createStart         | 创建时间-从         |            |                                                              |      |
| createEnd           | 创建时间-至         |            |                                                              |      |
| checkTimeStart      | 审核时间-从         |            |                                                              |      |
| checkTimeEnd        | 审核时间-至         |            |                                                              |      |
| turnFormalTimeStart | 转正时间-从         |            |                                                              |      |
| turnFormalTimeEnd   | 转正时间-至         |            |                                                              |      |
| applyStatus         | 申请状态            | Number数组 | 0:草稿 1:申请中 2:拒绝 3:已确认 4:已转正式 5:转正中 6:不合格 | 选填 |
| inviteStatus        | 邀约状态            | Number数组 | 0：未发起 1：已发起，对方未确认 2：对方已确认 3：邀约码失败 4:待采购审核 5:采购退回 | 选填 |

**当资质审核完成时间范围不为空获取数据状态：申请状态（apply_status）=已审核、邀约状态（invite_status ）= 已邀约 、供应商状态（vendor_status ）= 有效**

##### 1.1.5 说明

1、一次查询的 起-止时间范围： 24 小时内；

2、请求频率：不能小于 5 分钟
