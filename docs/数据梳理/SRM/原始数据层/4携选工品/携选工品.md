## 1 查询商品列表

  查询商品的列表。

### 1.1 请求

#### 1.1.1 请求地址

  https://openapi.xiekeyun.com/mro/sku/list

#### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

#### 1.1.3 业务请求参数（body）

| 参数名称 | 参数描述          | 参数类型   | 说明                                                         | 要求     |
| -------- | ----------------- | ---------- | ------------------------------------------------------------ | -------- |
| erpCode  | 请求者用户ERP帐号 | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，返回此员工有权限的数据 | **必填** |
| pageNum  | 页码              | Number     | 页码                                                         | **必填** |
| pageSize | 页面数量          | Number     | 页面数量                                                     | **必填** |

请求参数示例

        "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",

  本接口访问频率不能低于 **5** 秒。

#### 1.2.1 返回结果

##### 1.2.1.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型     |
| --------- | ------------------------------------------------------------ | -------- |

| errorMsg  | 接口错误信息（失败时有值）                                   | String   |
| data      | 接口成功时(result=1)返回                                     | Json对象 |

## 1 查询运费模板列表

  查询运费模板的列表。

### 1.1 请求

#### 1.1.1 请求地址

  https://openapi.xiekeyun.com/mro/freight/template/list

#### 1.1.2 请求公传参数（commonParam）

  **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

#### 1.1.3 业务请求参数（body）

#### 1.1.4 说明

  本接口访问频率不能低于 **5** 秒。

### 1.2 结果返回

#### 1.2.1 返回结果

##### 1.2.1.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型     |
| --------- | ------------------------------------------------------------ | -------- |
| result    | 返回结果状态1:成功；9:失败                                   | Number   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | String   |
| errorMsg  | 接口错误信息（失败时有值）                                   | String   |
| data      | 接口成功时(result=1)返回                                     | Json对象 |

##### 1.2.1.2 JSON对象(data)字段定义说明

运费计算将从最小梯度开始,总商品金额大于该值的场合,进入下一梯度计算。超过所有梯度的场合，视为免运费。如：梯度1：cost:6, value:100....梯度2：cost:8,value:50....总金额 = 40的场合运费8元，总金额80的场合运费6元，总金额120的场合运费0元

| 字段名称    | 描述                                                         | 类型       |
| ----------- | ------------------------------------------------------------ | ---------- |
| tempId      | 运费模板ID                                                   | String     |
| enable      | 是否生效中,只有一个会生效.当不存在生效的运费模板,将视为免运费 | Boolean    |
| list        | 运费梯度                                                     | Array      |
| list.lineNo | 梯度项次                                                     | Number     |
| list.cost   | 所需运费                                                     | BigDecimal |
| list.value  | 阈值                                                         | BigDecimal |
