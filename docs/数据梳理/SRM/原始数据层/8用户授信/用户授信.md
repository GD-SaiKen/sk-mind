# 使用场景描述

  用户（携客云账户体系）在携客云平台跳转到第三方网站时，第三方可以根据请求链接中的相关参数做验签；

  验签成功后，可以做如下动作：

   \1. 授信此用户是携客云平台的合法用户，并允许在本网站做相应的操作；

   \2. 获取此用户的相关信息；

# 3 参数说明

| 参数名称    | 参数描述       | 说明                                                         | 要求     |
| ----------- | -------------- | ------------------------------------------------------------ | -------- |
| xkyCompCode | 企业携客号     | 第三方企业在携客云平台的携客号                               | **必填** |
| xkyTime     | 时间           | 携客云平台生成授信的时间，值是时间戳数值；此值提供给第三方做时效性管控，防止他人使用同一个链接多次访问第三方平台。 | **必填** |
| checkToken  | 数据校验摘要值 | 对xkyCompCode、xkyTime原始数据生成摘要，防止数据被篡改；相应的摘要生成后续文档中详细说明。 | **必填** |
| ssoToken    | 密文字符串     | 携客云平台的用户信息加密后传给第三方的密文字符串，**使用Base64进行编码**，相应的解密方式后续文档中详细说明。 | **必填** |

# 4 摘要说明checktoken

## 4.1 生成方式

  checkToken是携客云平台针对原始参数生成的摘要，其生成方式如下：

| 指标     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 摘要算法 | MD5                                                          |
| 原始数据 | 生成摘要的原始数据包括3部分： 1. 参数xkyCompCode对应企业在携客云平台的appKey； 2. 参数xkyCompCode的值； 3. 参数xkyTime的值；三个参数值使用下划线(_)分隔；参数值的拼接顺序为： 根据参数名称的字典顺序排列后取其值。（即 appkey，xkyCompCode，xkyTime） |
| 摘要长度 | 32位                                                         |

## 4.2 摘要验证

  根据**【参数说明】**的描述，生成出来的摘要字符串与请求传入的checkToken的内容做比较。

  如果一致，摘要验证成功；如果不一致，说明原始数据被篡改，第三方根据自身业务进行处理。

## 4.3 生成样例

| 描述             | 内容                                                         |
| ---------------- | ------------------------------------------------------------ |
| 参数             | xkyCompCode=61578989xkyTime=1545036456270appKey=e83be93cff7128886279415b96a5ff68 |
| 拼接的原始字符串 | e83be93cff7128886279415b96a5ff68_61578989_1545036456270      |
| 生成摘要         | MD5('e83be93cff7128886279415b96a5ff68_61578989_1545036456270') |
| **摘要内容**     | **171c5f2b8c7e17d8cb30d75255efd08b**                         |

# 6 ssotoken解密

  如需查看ssoToken的原始内容，请使用如下方式进行解密：

| 指标     | 说明                                                        |
| -------- | ----------------------------------------------------------- |
| 解密算法 | AES                                                         |
| 解密模式 | ECB                                                         |
| 填充方式 | PKCS5Padding                                                |
| 数据块   | 128位                                                       |
| 密钥     | 参数xkyCompCode对应企业在携客云平台的secretKey              |
| 解码格式 | **Base64** **（在解密时，先使用Base64对ssoToken进行解码）** |
| 字符集   | UTF-8                                                       |

## 说明

```java
        /*
         * 如果您有Java开发环境，可以直接使用以下代码逻辑转换
         */
        public static void main(String[] args) throws Exception{
            String key = "234skjlsjsdfisdpgsgdsg";//企业密钥
            // 生成Key
            KeyGenerator keyGenerator = KeyGenerator.getInstance("AES");
            keyGenerator.init(128, new SecureRandom(key.getBytes()));
            SecretKey secretKey = keyGenerator.generateKey();
            byte[] keyBytes = secretKey.getEncoded();
            sun.misc.BASE64Encoder coder = new sun.misc.BASE64Encoder();
            // 此结果就是C#解密时的密钥
            System.out.println(coder.encode(keyBytes));
        }
```

## java版sdk

代码

```java
 /**
     * @Description: 根据企业的secretKey对字符串解密
     * @param compSecretKey  企业的密钥
     * @param pwdStr  需要解密的字符串
     * @return String
     */
    public static String jdkAESDec(String compSecretKey, String pwdStr) {
        try {
            Cipher cipher = Cipher.getInstance("AES/ECB/PKCS5Padding");
            // 解密
            cipher.init(Cipher.DECRYPT_MODE,createAESKey(compSecretKey));
            byte[] decodeResult = cipher.doFinal(Base64.getDecoder().decode(pwdStr));
            String src=new String(decodeResult,"UTF-8");
            return src;
        } catch (Exception e) {
            e.printStackTrace();
        }
        return "";
    }

/**
     * @Description: 生成AES的Key对象
     * @param compSecretKey  企业的secretKey
     * @return Key
     */
    public static Key createAESKey(String compSecretKey) throws Exception{
        // 生成Key
        KeyGenerator keyGenerator = KeyGenerator.getInstance("AES");
        SecureRandom secureRandom = SecureRandom.getInstance("SHA1PRNG");
        secureRandom.setSeed(compSecretKey.getBytes());
        keyGenerator.init(128, secureRandom);
        SecretKey secretKey = keyGenerator.generateKey();
        byte[] keyBytes = secretKey.getEncoded();
        // Key转换
        return new SecretKeySpec(keyBytes, "AES");
    }
```
