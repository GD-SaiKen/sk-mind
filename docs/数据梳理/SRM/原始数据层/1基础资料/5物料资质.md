## 1 物料资质列表查询

   此API接口是查询平台物料资质列表数据

## 1.1 请求

### 1.1.1 请求地址

[https://openapi.xiekeyun.com/prodCertificates/list.json](https://openapi.xiekeyun.com/prodCertificates/list.json)

### 1.1.2 请求公传参数（commonParam）

   **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E5%85%AC%E4%BC%A0%E5%8F%82%E6%95%B0)】所述**

### 1.1.3 业务请求参数（body）

|参数名称|参数描述|参数类型|说明|要求|
|---|---|---|---|---|
|erpCode|请求者用户ERP帐号|**String**|对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据|**必填**|
|prodCodeList|物料编码列表|**List**|物料编码，一次请求最多100个物料|**必填**|

### 1.1.5 请求参数示例

Demo

    `{         "commonParam": {                 "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",                 "version": "1.0",                 "operateCompanyCode": "91904378", "sign": "989ec1f2e7fe0bdfc0d359331c027159", "timestamps": "1666067977"         },  "body": {   "erpCode": "6666",   "prodCodeList":["IP03000021","LM30604.0067","M302000019"]  } }`
    

### 1.1.6 说明

   1. 本接口访问频率不能低于6秒；

## 1.2 结果返回

### 1.2.1 描述

结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

|字段名称|描述|类型|
|---|---|---|
|result|返回结果状态 1:成功；9:失败|**Number**|
|errorCode|接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E9%94%99%E8%AF%AF%E7%BC%96%E7%A0%81%E8%A7%84%E8%8C%83%E5%AE%9A%E4%B9%89)”|**String**|
|errorMsg|接口错误信息（失败时有值）|**String**|
|dataList|物料和资质列表对象集合|**Json数组**|

### 1.2.2.2 dataList集合对应一个物料所有的资质数据

|字段名称|描述|类型|
|---|---|---|
|prodCode|物料编码|**String**|
|certificatesList|资质列表|**Json数组**|

### 1.2.2.3 dataList集合对应一个资质数据

|字段名称|描述|类型|
|---|---|---|
|vcXkNo|平台单据号|**String**|
|innerVendorName|内部供应商名称|**String**|
|innerVendorCode|供应商erp编码|**String**|
|certTitle|资质标题|**String**|
|certImportance|重要程度，0 可选，1 必须|**String**|
|validType|有效类型 0: 长期 1: 一年 2: 两年 3: 三年 4: 指定,5:半年|**Number**|
|validTime|有效期 （时间戳）|**Number**|
|validCount|有效数量|**Number**|
|uploadType|上传类型 0 采购上传 1 供应商上传|**String**|
|attachCount|附件数量|**Number**|
|status|资质状态 0：草稿 1：待回复 2：待确认 3：退回待回复 4：已完成 5：作废 6：撤回|**Number**|
|dueDateIsMust|资质文件到期日是否必填 0：不是必填，1：必填|**Number**|
|dueDate|资质文件到期日（时间戳）|**Number**|
|documentCode|文件编码|**String**|
|qualificationFileCode|资质文件编码|**String**|
|subimTime|提交时间 （时间戳）|**Number**|
|issuingInstitution|发证机构|**String**|
|certificationStandards|认证标准|**String**|
|platformExtendN01|平台扩展字段1|**String**|
|platformExtendN02|平台扩展字段2|**String**|
|platformExtendN03|平台扩展字段3|**String**|
|platformExtendN04|平台扩展字段4|**String**|
|platformExtendN05|平台扩展字段5|**String**|
|platformExtendN06|平台扩展字段6|**String**|

## 2 查询物料资质有异动的物料列表

   此API接口是查询平台物料资质列表数据

## 2.1 请求

### 2.1.1 请求地址

[https://openapi.xiekeyun.com/prodCertificates/fluctuationProdList.json](https://openapi.xiekeyun.com/prodCertificates/fluctuationProdList.json)

### 2.1.2 请求公传参数（commonParam）

   **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E5%85%AC%E4%BC%A0%E5%8F%82%E6%95%B0)】所述**

### 2.1.3 业务请求参数（body）

|参数名称|参数描述|参数类型|说明|要求|
|---|---|---|---|---|
|erpCode|请求者用户ERP帐号|**String**|对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据|**必填**|
|startDate|修改时间|**Number**|开始时间对应的**毫秒**时间戳数值; 查询时为开区间（即 >startDate）|**必填**|
|endDate|修改时间|**Number**|结束时间对应的**毫秒**时间戳数值; 查询时为闭区间（即 <=endDate）(startDate,endDate]最大时间区间为3天|**必填**|
|status|单据状态|**List**|状态 0：草稿 1：待回复 2：待确认 3：退回待回复 4：已完成 5：作废 6：撤回|**选填**|

### 2.1.5 请求参数示例

Demo

    `{   "commonParam": {     "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",     "version": "1.0",     "ownerCompanyCode": "91904378",     "operateCompanyCode": "91904378",     "sign": "75926cb4678d9fce912014de049ca0a6",     "timestamps": "1757476693"   },   "body": {     "erpCode": "6666",     "startDate": "1757433600000",     "endDate": "1757519999000",     "statusList": [2]   } }`
    

### 2.1.6 说明

   1. 本接口访问频率不能低于6秒；

## 2.2 结果返回

### 2.2.1 描述

结果数据以Json格式返回

### 2.2.2 返回结果

### 2.2.2.1 返回结果公共字段定义

|字段名称|描述|类型|
|---|---|---|
|result|返回结果状态 1:成功；9:失败|**Number**|
|errorCode|接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E9%94%99%E8%AF%AF%E7%BC%96%E7%A0%81%E8%A7%84%E8%8C%83%E5%AE%9A%E4%B9%89)”|**String**|
|errorMsg|接口错误信息（失败时有值）|**String**|
|dataList|物料编码对象集合|**Json数组**|

### 2.2.2.2 dataList集合对应一个物料所有的资质数据

|字段名称|描述|类型|
|---|---|---|
|prodCode|物料编码|**String**|

### 2.2.3 查询返回示例

    {   "result": 1,   "dataList": [     {       "prodCode": "000-260200-001H"     }   ],   "data": null }
