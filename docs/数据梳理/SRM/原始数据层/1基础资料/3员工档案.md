## 1 员工档案列表查询

   此API接口是查询平台员工列表数据

## 1.1 请求

### 1.1.1 请求地址

[https://openapi.xiekeyun.com/employee/list.json](https://openapi.xiekeyun.com/delivery/getNoList.json)

### 1.1.2 请求公传参数（commonParam）

   **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E5%85%AC%E4%BC%A0%E5%8F%82%E6%95%B0)】所述**

### 1.1.3 业务请求参数（body）

|参数名称|参数描述|参数类型|说明|要求|
|---|---|---|---|---|
|erpCode|请求者用户ERP帐号|**String**|对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据|**必填**|
|inviteStatus|邀约状态|**Number**|邀约状态：  <br>0：未发起  <br>1：已发起，未处理  <br>2：已加入  <br>3：邀约失效  <br>4：离职|**选填**|
|returnTransObject|是否返回交易对象列表 （0：不返回， 1：返回），默认=0|**Number**||**选填**|

### 1.1.5 请求参数示例

Demo

    `{     "commonParam": {         "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",         "version": "1.0",         "ownerCompanyCode": "91904378",         "operateCompanyCode": "91904378",         "sign": "ba80170df56415d2bcf478b2fc69b70d",         "timestamps": "1759214961"     },     "body": {         "erpCode": "69999",         "inviteStatus": 2,         "returnTransObject": 1     } }`
    

### 1.1.6 说明

   1. 本接口访问频率不能低于1分钟；

## 1.2 结果返回

### 1.2.1 描述

结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

|字段名称|描述|类型|
|---|---|---|
|result|返回结果状态 1:成功；9:失败|**Number**|
|errorCode|接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E9%94%99%E8%AF%AF%E7%BC%96%E7%A0%81%E8%A7%84%E8%8C%83%E5%AE%9A%E4%B9%89)”|**String**|
|errorMsg|接口错误信息（失败时有值）|**String**|
|dataList|员工档案集合|**Json数组**|

### 1.2.2.2 dataList集合对应一个员工档案数据

|字段名称|描述|类型|
|---|---|---|
|companyCode|企业编码|**String**|
|xkUniqueCode|员工平台编码|**String**|
|employeeName|员工姓名|**String**|
|employeeNumber|工号|**String**|
|departmentName|部门名称|**String**|
|mobile|手机号|**String**|
|mobileArea|手机地区码|**String**|
|email|邮箱|**String**|
|erpAccount|ERP帐号|**String**|
|erpEmployeeCode|ERP员工编码|**String**|
|employeeFlag|员工状态 1：正常 0：停用|**Number**|
|userType|用户类型 0：普通用户 1：管理员|**Number**|
|loginName|携客号|**String**|
|userName|用户名|**String**|
|roleCode|岗位编码|**String**|
|roleName|岗位名称|**String**|
|inviteStatus|邀约状态 0：未发起 1：已发起，未处理 2：已发起已处理 3：邀约码失效 4：离职|**Number**|
|inviteTime|邀约时间（时间戳）|**Number**|
|remark|备注|**String**|
|transObjectList|交易对象列表|**Json数组**|

### 1.2.2.3 transObjectList集合对应一个交易对象数据

|字段名称|描述|类型|
|---|---|---|
|transInnerCode|内部供应商编码|**String**|
|sourceBelongs|供应商类型: 0:正式， 1:潜在|**Number**|
