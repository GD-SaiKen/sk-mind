## 1 岗位权限列表查询

   此API接口是查询平台岗位权限列表数据

## 1.1 请求

### 1.1.1 请求地址

[https://openapi.xiekeyun.com/sysRole/list.json](https://openapi.xiekeyun.com/sysRole/list.json)

### 1.1.2 请求公传参数（commonParam）

   **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E5%85%AC%E4%BC%A0%E5%8F%82%E6%95%B0)】所述**

### 1.1.3 业务请求参数（body）

|参数名称|参数描述|参数类型|说明|要求|
|---|---|---|---|---|
|erpCode|请求者用户ERP帐号|**String**|对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据|**必填**|

### 1.1.5 请求参数示例

Demo

    `{             "commonParam": {                 "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",                 "version": "1.0",                 "ownerCompanyCode": "91904378",                 "operateCompanyCode": "91904378",                 "sign": "ce9da88606be49d6ee731862c67417d1",                 "timestamps": "1679293719"             },             "body": {                 "erpCode": "dh888888"             }         }`
    

### 1.1.6 说明

   1. 本接口访问频率不能低于1分钟；

## 1.2 结果返回

### 1.2.1 描述

结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

|字段名称|描述|类型|
|---|---|---|
|result|返回结果状态 1:成功；9:失败|**Number**|
|errorCode|接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E9%94%99%E8%AF%AF%E7%BC%96%E7%A0%81%E8%A7%84%E8%8C%83%E5%AE%9A%E4%B9%89)”|**String**|
|errorMsg|接口错误信息（失败时有值）|**String**|
|dataList|岗位权限集合|**Json数组**|

### 1.2.2.2 dataList集合对应一个员工档案数据

|字段名称|描述|类型|
|---|---|---|
|roleCode|岗位编码|**String**|
|roleName|岗位名称|**String**|
|departmentName|部门名称|**String**|
|parentRoleId|上级岗位ID|**String**|
|parentRoleCode|上级岗位编码|**String**|
|parentRoleName|上级岗位名称|**String**|
|enableFlag|启用状态 0：不启用 1：启用 默认为：1|**String**|
