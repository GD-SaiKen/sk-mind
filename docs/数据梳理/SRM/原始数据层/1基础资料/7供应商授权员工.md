## 1 供应商授权员工列表查询

此API接口是查询平台供应商授权员工列表数据

## 1.1 请求

### 1.1.1 请求地址

https://openapi.xiekeyun.com/assignVTrans/list.json

### 1.1.2 请求公传参数（commonParam）

**请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称            | 参数描述          | 参数类型       | 说明                                                         | 要求     |
| ------------------- | ----------------- | -------------- | ------------------------------------------------------------ | -------- |
| erpCode             | 请求者用户ERP帐号 | **String**     | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据 | **必填** |
| innerVendorCodeList | 供应商编码集合    | **String数组** | **一次请求中，至少需要包含一个供应商编码、最大个数为100；另外起止更新时间与供应商编码不能存在同时为空** 如果超过系统不作处理并返回相应错误信息 | 选填     |
| roleCode            | 岗位权限编码      | **String**     | 10004 :业务员; 10006 :来料质检; 10010 :仓库出货; 10012 :应收对账; | 选填     |
| updateStartDate     | 更新开始时间      | **Number**     | 开始时间对应的**毫秒**时间戳数值; 查询时为开区间（即 >updateStartDate） | 选填     |
| updateEndDate       | 更新结束时间      | **Number**     | 结束时间对应的**毫秒**时间戳数值; 查询时为开区间（即 <updateEndDate） | 选填     |
| authStatus          | 授权状态          | **Integer**    | 1：有效 2：失效 3：停用                                      | 选填     |

## 1.2 结果返回

### 1.2.1 描述

结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 当前资质列表对象集合                                         | **Json数组** |

### 1.2.2.2 dataList集合对应供应商当前资质数据

| 字段名称            | 描述                                                         | 类型        |
| ------------------- | ------------------------------------------------------------ | ----------- |
| employeeName        | 标题                                                         | **String**  |
| mobileArea          | 手机地区码                                                   | **String**  |
| mobile              | 手机号                                                       | **String**  |
| email               | 邮箱                                                         | **String**  |
| verifyEmail         | 校验邮箱                                                     | **String**  |
| authEmployeeName    | 授权人员工姓名                                               | **String**  |
| authTime            | 授权时间                                                     | **Number**  |
| authStatus          | 授权状态 1：有效、2：失效：3：停用                           | **Integer** |
| disableEmployeeName | 停用人员工姓名                                               | **String**  |
| disableTime         | 停用时间                                                     | **Number**  |
| roleCode            | 岗位类型权限码10004:业务员；10006:来料质检；10010:仓库出货；10012: 应收对账 | **String**  |
| roleName            | 岗位类型权限名称                                             | **String**  |
| innerVendorCode     | 供应商编码                                                   | **String**  |
