## 1 物料申请单列表查询

此查询API接口是查找物料申请单列表信息

## 1.1 请求

### 1.1.1 请求地址

https://openapi.xiekeyun.com/prodApply/list.json

### 1.1.2 请求公传参数（commonParam）

**请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称        | 参数描述          | 参数类型   | 说明                                                         | 要求     |
| --------------- | ----------------- | ---------- | ------------------------------------------------------------ | -------- |
| erpCode         | 请求者用户ERP帐号 | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，用于判断此员工在平台是否有此数据权限 | **必填** |
| createTimeStart | 创建时间          | **Number** | 开始时间对应的时间戳数值; 查询时为开区间（即 >createStart）  | **必填** |
| createTimeEnd   | 创建时间          | **Number** | 结束时间对应的时间戳数值; 查询时为闭区间（即 <=createEnd）(createStart,createEnd]最大时间区间为30天 | **必填** |
| status          | 状态              | **Number** | 状态: 0:待发布 1:待审核 2:已拒绝 3:已完成                    | **选填** |

### 1.1.4 请求参数示例

Demo

```
{
  "commonParam": {
    "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",
    "version": "1.0",
    "ownerCompanyCode": "91904378",
    "operateCompanyCode": "91904378",
    "sign": "e6f12bd68b0b154abbeaab8fb3414d15",
    "timestamps": "1718363630"
  },
  "body": {
    "erpCode": "6666",
    "createTimeStart": "1718010940000",
    "createTimeEnd": "1718356540000"
  }
}
    
```

### 1.1.5 说明

\1. 本接口访问频率不能低于 **5** 分钟；

\2. 一次请求中开始结束时间范围：不能大于 **30** 天

## 1.2 结果返回

### 1.2.1 描述

结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 物料申请单数据列表                                           | **JSON数组** |

### 1.2.2.2 dataList字段定义

| 字段名称 | 描述                                         | 类型       |
| -------- | -------------------------------------------- | ---------- |
| cpaXkNo  | 携客云平台单号                               | **String** |
| prodName | 物料名称                                     | **String** |
| status   | 状态: 0:待发布、1:待审核、2:已拒绝、3:已完成 | **Number** |

| erpCode  | 请求者用户ERP帐号 | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用 | **必填** |
