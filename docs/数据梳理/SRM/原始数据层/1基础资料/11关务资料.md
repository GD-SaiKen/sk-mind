## 1 关务资料列表查询

此查询API接口是查找关务资料列表信息

## 1.1 请求

### 1.1.1 请求地址

https://openapi.xiekeyun.com/prodCustoms/list.json

### 1.1.2 请求公传参数（commonParam）

**请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称        | 参数描述          | 参数类型       | 说明                                                         | 要求     |
| --------------- | ----------------- | -------------- | ------------------------------------------------------------ | -------- |
| erpCode         | 请求者用户ERP帐号 | **String**     | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，用于判断此员工在平台是否有此数据权限 | **必填** |
| startTime       | 修改时间          | **Number**     | 开始时间对应的时间戳毫秒数值; 查询时为开区间（即 >startTime） | **必填** |
| endTime         | 修改时间          | **Number**     | 结束时间对应的时间戳毫秒数值; 查询时为闭区间（即 <=endTime） | **必填** |
| statusList      | 状态              | **Number数组** | 状态: 0:未完成 1:采购审核 2:已完成 3:退回 4:已撤回 5:待回复 6:关务审核 7:作废 | **选填** |
| prodCode        | 物料编码          | **String**     |                                                              | **选填** |
| innerVendorCode | 供应商编码        | **String**     |                                                              | **选填** |

### 1.1.4 请求参数示例

Demo

```
{
  "commonParam": {
    "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",
    "version": "1.0",
    "ownerCompanyCode": "91904378",
    "operateCompanyCode": "91904378",
    "sign": "15e9659a3e49a5df7bdab3d4c3ada1a3",
    "timestamps": "1758878976"
  },
  "body": {
    "erpCode": "6666",
    "startTime": "1758817775000",
    "endTime": "1758900575000"
  }
}
    
```

### 1.1.5 说明

\1. 本接口访问频率不能低于 **5** 分钟；

\2. 一次请求中开始结束时间范围：不能大于 **24** 小时；

\3. 查询数据时，不能查询 **7** 天之前数据

## 1.2 结果返回

### 1.2.1 描述

结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 关务资料数据列表                                             | **JSON数组** |

### 1.2.2.2 dataList字段定义

| 字段名称        | 描述                                                         | 类型       |
| --------------- | ------------------------------------------------------------ | ---------- |
| cpcXkNo         | 携客云平台单号                                               | **String** |
| innerVendorCode | 供应商编码                                                   | **String** |
| prodCode        | 物料编码                                                     | **String** |
| status          | 状态: 0:未完成,1:采购审核,2:已完成,3:退回,4:已撤回,5:待回复,6:关务审核,7:作废 | **Number** |
