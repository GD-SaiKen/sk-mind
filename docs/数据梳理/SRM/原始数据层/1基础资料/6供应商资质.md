## 1 查询资质有修改的供应商列表

此API接口是查询平台供应商当前资质列表数据

## 1.1 请求

### 1.1.1 请求地址

https://openapi.xiekeyun.com/vendorCertificates/queryVendorListByUpdateTime.json

### 1.1.2 请求公传参数（commonParam）

**请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 1.1.3 业务请求参数（body）

| 参数名称   | 参数描述                                                     | 参数类型   | 说明                                                         | 要求     |
| ---------- | ------------------------------------------------------------ | ---------- | ------------------------------------------------------------ | -------- |
| erpCode    | 请求者用户ERP帐号                                            | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据 | **必填** |
| startTime  | 开始时间对应的时间戳数值; 查询时为开区间（即 >startTime）    | **Number** | 对应的最新修改时间                                           | **必填** |
| endTime    | 结束时间对应的时间戳数值; 查询时为闭区间（即 <=endTime）(startTime,endTime]最大时间区间为30天 | **Number** |                                                              | **必填** |
| vendorType | 供应商类型: 1:正式供应商;2:潜在供应商                        | **Number** | 默认是正式供应商                                             | 选填     |
| status     | 资质状态 0：草稿 1：待回复 2：待确认 3：退回待回复 4：已完成 5：作废 6：撤回 | **Number** |                                                              | 选填     |

### 1.1.4 请求参数示例

Demo

```
{
  "commonParam":{
    "appKey":"3da512e5eb0aa2fa27b6c3ffc638da22",
    "version":"1.0",
    "ownerCompanyCode":"91904378",
    "operateCompanyCode":"91904378",
    "sign": "3b40c7cc2ead5e5c9de077066541eca4",
    "timestamps": "1730273650"
  },
  "body":{
    "erpCode":"6666",
    "startTime":1729353600000,
    "endTime":1730304000000,
    "status": 2
  }
}
    
```

### 1.1.5 说明

\1. 本接口访问频率不能低于60秒；

## 1.2 结果返回

### 1.2.1 描述

结果数据以Json格式返回

### 1.2.2 返回结果

### 1.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 当前资质供应商列表对象集合                                   | **Json数组** |

### 1.2.2.2 dataList集合对应供应商当前资质数据

| 字段名称           | 描述               | 类型       |
| ------------------ | ------------------ | ---------- |
| innerVendorCode    | 供应商编码         | **String** |
| innerVendorName    | 供应商名称         | **String** |
| vendorXkUniqueCode | 供应商平台唯一编码 | **String** |

## 2 一个供应商资质列表查询

此API接口是查询平台一个供应商当前资质列表数据

## 2.1 请求

### 2.1.1 请求地址

https://openapi.xiekeyun.com/vendorCertificates/list.json

### 2.1.2 请求公传参数（commonParam）

**请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

### 2.1.3 业务请求参数（body）

| 参数名称           | 参数描述                              | 参数类型   | 说明                                                         | 要求                                          |
| ------------------ | ------------------------------------- | ---------- | ------------------------------------------------------------ | --------------------------------------------- |
| erpCode            | 请求者用户ERP帐号                     | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据 | **必填**                                      |
| vendorXkUniqueCode | 内部供应商平台编码                    | **String** | 内部供应商平台编码，系统生成的32位唯一编码                   | **内部供应商平台编码和供应商erp编码必填一个** |
| innerVendorCode    | 供应商erp编码                         | **String** |                                                              | **内部供应商平台编码和供应商erp编码必填一个** |
| vendorType         | 供应商类型: 1:正式供应商;2:潜在供应商 | **Number** | 默认是正式供应商                                             | 选填                                          |
| startTime          | 开始时间对应的时间戳数值              | **Number** | 对应的最新修改时间                                           | 选填                                          |
| endTime            | 结束时间对应的时间戳数值              | **Number** |                                                              | 选填                                          |

### 2.1.4 请求参数示例

Demo

```
{
    "commonParam": {
        "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",
        "version": "1.0",
        "ownerCompanyCode": "91904378",
        "operateCompanyCode": "91904378",
        "sign": "c01a4b4915d7836c339ea15021874d30",
        "timestamps": "1667808586"
    },
    "body": {
        "erpCode": "6666",
        "innerVendorCode": "FMT12"
    }
}
    
```

### 2.1.5 说明

\1. 本接口访问频率不能低于60秒；

## 2.2 结果返回

### 2.2.1 描述

结果数据以Json格式返回

### 2.2.2 返回结果

### 2.2.2.1 返回结果公共字段定义

| 字段名称  | 描述                                                         | 类型         |
| --------- | ------------------------------------------------------------ | ------------ |
| result    | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg  | 接口错误信息（失败时有值）                                   | **String**   |
| dataList  | 当前资质列表对象集合                                         | **Json数组** |

### 2.2.2.2 dataList集合对应供应商当前资质数据

| 字段名称              | 描述                                                         | 类型       |
| --------------------- | ------------------------------------------------------------ | ---------- |
| certTitle             | 标题                                                         | **String** |
| qualificationFileCode | 资质文件编码                                                 | **String** |
| certificatesTypeName  | 资质分类                                                     | **String** |
| certImportance        | 是否必要（null/0：否 1：是）                                 | **String** |
| uploadType            | 提供方（0：本方（采购方）上传（对应文件操作为上传附件） 1：供应商上传（对应文件操作为邀请供应商）） | **String** |
| dueDate               | 文件到期日                                                   | **Number** |
| createTime            | 建立日期                                                     | **Number** |
| validTime             | 到期日                                                       | **Number** |
| validType             | 有效期 0: 长期 1: 一年 2: 两年 3: 三年 4: 指定,5:半年        | **Number** |
| validCount            | 有效数量                                                     | **Number** |
| documentCode          | 文件编码                                                     | **String** |
| status                | 资质状态 0：草稿 1：待回复 2：待确认 3：退回待回复 4：已完成 5：作废 6：撤回 | **Number** |
| vcXkNo                | 资质文件平台单号                                             | **String** |
| certAuditName         | 资质审核名称                                                 | **String** |
