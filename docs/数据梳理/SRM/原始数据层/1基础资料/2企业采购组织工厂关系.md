# 企业采购组织工厂关系

## 企业集成的采购组织列表查询

    此API接口是查询企业集成的采购组织+工厂列表数据。

### 1 请求

#### 1.1 请求地址

    [https://openapi.xiekeyun.com/compOrg/itgOrgPlants.json](https://openapi.xiekeyun.com/compOrg/itgOrgPlants.json)

#### 1.2 请求公传参数（commonParam）

     **请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E5%85%AC%E4%BC%A0%E5%8F%82%E6%95%B0)】所述**

#### 1.3 业务请求参数（body）

|参数名称|参数描述|参数类型|说明|要求|
|---|---|---|---|---|
|erpCode|请求者用户ERP帐号|**String**|对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，创建此员工写入的数据|**必填**|

#### 1.4 请求参数示例

Demo

    `{     "commonParam":{         "appKey":"3da512e5eb0aa2fa27b6c3ffc638da22",         "version":"1.0",         "ownerCompanyCode":"91904378",         "operateCompanyCode":"91904378",         "sign": "17d69cc937078a7767c2382d1b3ced25",         "timestamps": "1615280678"     },     "body":{         "erpCode":"00001"     } }`
    

#### 1.5 说明

​ 1. 本接口访问频率不能低于 **5** 分钟；

### 2 结果返回

#### 2.1 描述

​ 结果数据以Json格式返回

#### 2.2 返回结果

#### 2.2.1 返回结果返回结果字段定义

|字段名称|描述|类型|
|---|---|---|
|result|返回结果状态 1:成功；9:失败|**Number**|
|errorCode|接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#%E9%94%99%E8%AF%AF%E7%BC%96%E7%A0%81%E8%A7%84%E8%8C%83%E5%AE%9A%E4%B9%89)”|**String**|
|errorMsg|接口错误信息（失败时有值）|**String**|
|dataList|已集成的采购组织+工厂列表数据|**JSON数组**|

#### 2.2.2 dataList中一个Json数据明细字段定义

|字段名称|描述|类型|
|---|---|---|
|erpOrgCode|ERP中采购组织编码|**String**|
|erpOrgName|ERP中采购组织名称|**String**|
|erpPlantCode|ERP中工厂编码|**String**|
|erpPlantName|ERP中工厂名称|**String**|
