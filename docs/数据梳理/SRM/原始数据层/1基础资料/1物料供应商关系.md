## 1 企业物料供应商交货方式查询

此查询API接口是第三方系统(ERP)按物料和供应商从平台读取交货方式。

### 1.1 请求

#### 1.1.1 请求地址

https://openapi.xiekeyun.com/product/vendor/receiveTypes.json

#### 1.1.2 请求公传参数（commonParam）

**请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

#### 1.1.3 业务请求参数（body）

| 参数名称        | 参数描述          | 参数类型   | 说明                                                         | 要求     |
| --------------- | ----------------- | ---------- | ------------------------------------------------------------ | -------- |
| prodCodes       | 物料编码列表      | **List**   | 多个物料编码批量推到携客云平台，一次请求中，至少需要包含一条数据、包含的数据个数最大为**500，**如果超过系统不作处理并返回相应错误信息 | **选填** |
| erpCode         | 请求者用户ERP帐号 | **String** | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，用于判断此员工在平台是否有此数据权限 | **必填** |
| innerVendorCode | 内部供应商编码    | **String** | 供应商档案中的供应商编码，查询时，传入的值应该与采购单中的供应商一致 | **必填** |

#### 1.1.4 请求参数示例

Demo

```
{
  "commonParam":{
    "appKey":"8beef1c48f31b188d989b7acff715f0a",
    "version":"1.0",
    "sign":"6f382ee57f6b133b5a5f02c76ce9bf9a",
    "timestamps":1555072169,
    "ownerCompanyCode":"70267016",
    "operateCompanyCode":"70267016"
  },
  "body":{
    "erpCode":"wtest",
    "innerVendorCode":"ACT01",
    "prodCodes":[
      "01010000021",
      "190204030024"
    ]
  }
}
    
```

#### 1.1.5 说明

本接口查询同一个供应商时，访问频率不能低于 **1** 分钟；

### 1.2 结果返回

#### 1.2.1 描述

结果数据以Json格式返回

#### 1.2.2 返回结果

##### 1.2.2.1 返回结果公共字段定义

| 字段名称     | 描述                                                         | 类型         |
| ------------ | ------------------------------------------------------------ | ------------ |
| result       | 返回结果状态 1:成功；9:失败                                  | **Number**   |
| errorCode    | 接口错误编码（失败时有值），详细参考“[**错误编码规范定义**](https://openapi.xiekeyun.com/doc/doccommon/description.html#错误编码规范定义)” | **String**   |
| errorMsg     | 接口错误信息（失败时有值）                                   | **String**   |
| **dataList** | 返回数据                                                     | **Json对象** |

##### 1.2.2.2 返回数据 (dataList)一个对象的字段定义说明

| 字段名称        | 描述                             | 类型        |
| --------------- | -------------------------------- | ----------- |
| productCode     | 物料编码                         | **String**  |
| innerVendorCode | 内部供应商编码                   | **String**  |
| receiveType     | 收货方式（1：按订单；2：按排程） | **Integer** |

#### 1.2.3 查询返回示例

Demo

```
{
  "result":1,
  "dataList":[
    {
      "productCode":"01010000021",
      "innerVendorCode":"ACT01",
      "receiveType":**1**
    },
    {
      "productCode":"190204030024",
      "innerVendorCode":"ACT01",
      "receiveType":**1**
    }
  ]}
    
```

## 2 根据物料编码查询核定供应商

此查询API接口是第三方系统按物料获取核定供应商。

### 2.1 请求

#### 2.1.1 请求地址

https://openapi.xiekeyun.com/compProd/pvRelation/queryList.json

#### 2.1.2 请求公传参数（commonParam）

**请求时对应的公传参数定义如【[公传参数](https://openapi.xiekeyun.com/doc/doccommon/description.html#公传参数)】所述**

#### 2.1.3 业务请求参数（body）

| 参数名称         | 参数描述                                               | 参数类型     | 说明                                                         | 要求                               |
| ---------------- | ------------------------------------------------------ | ------------ | ------------------------------------------------------------ | ---------------------------------- |
| erpCode          | 请求者用户ERP帐号                                      | **String**   | 对应数据操作公司中，员工档案里“ERP帐户”的值，主要做数据鉴权使用，用于判断此员工在平台是否有此数据权限 | **必填**                           |
| prodCodeList     | 物料编码列表                                           | **Json数组** | 多个物料编码批量推到携客云平台，一次请求中，至少需要包含一条数据、包含的数据个数最大为**500，**如果超过系统不作处理并返回相应错误信息 | **物料编码列表和时间范围必填一项** |
| updateStart      | 最新修改开始时间                                       | **Number**   | 对应的时间戳数值;查询时为开区间（即 >= updateTime）          | **物料编码列表和时间范围必填一项** |
| updateEnd        | 最新修改结束时间                                       | **Number**   | 对应的时间戳数值;查询时为开区间（即 <= updateTime）(updateStart,updateEnd]最大时间区间为一天 | **物料编码列表和时间范围必填一项** |
| categoryCodeList | 分类编码列表                                           | **Json数组** |                                                              | **选填**                           |
| categoryType     | 分类类型:ERP分类:2000 分类1:2001 分类2:2002 分类3:2003 | **String**   | 分类编码列表不为空时必填                                     | **选填**                           |

#### 2.1.4 请求参数示例

Demo

```
{

 "appKey": "3da512e5eb0aa2fa27b6c3ffc638da22",

 "version": "1.0",

"sign": "e38d610d763a643f37e34e4c0a95e1d6",

"timestamps": "1657194136"

 },

 "body": {

 "erpCode": "6666",

 "prodCodeList":["ZJQREQTEST3"]

 }

}
    
```

#### 

#### 2.1.5 说明

本接口查询访问频率不能低于 **1** 分钟；

### 2.2 结果返回

#### 2.2.1 描述

结果数据以Json格式返回

#### 2.2.2 返回结果

##### 2.2.2.1 返回结果公共字段定义

| errorMsg     | 接口错误信息（失败时有值）                                   | **String**   |

| customsRecord        | 关务备案状态，0:未备案；1：已备案                            | **Number** |
| minPurchaseQty       | 最小采购量                                                   | **Number** |
| verifyFlag           | 检验否:1是，0否                                              | **Number** |
| brandName            | 品牌名称                                                     | **String** |
| manufacturerProdCode | 制造商料号                                                   | **String** |
| selectedType         | 中标类型枚举值：1大标、2小标、3独标、4主选、5备选、6平标、7自定配额 | **String** |
| outsourcingLossRate  | 委外损耗率（0.32，表示32%）                                  | **Number** |
| platformExtendN01    | 自定义字段1                                                  | **String** |
| platformExtendN02    | 自定义字段2                                                  | **String** |
