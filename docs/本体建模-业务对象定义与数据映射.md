## 第一部分：业务对象定义

### 对象分类体系

```
┌─────────────────────────────────────────────────────┐
│                    主数据对象 (Master Data)           │
│  Material/物料  Supplier/供应商  Customer/客户        │
│  Account/会计科目  Product/产品                      │
├─────────────────────────────────────────────────────┤
│                    交易对象 (Transaction)             │
│  PurchaseOrder/采购订单  SalesOrder/销售订单          │
│  RequisitionOrder/请购单  ReturnOrder/退货单          │
├─────────────────────────────────────────────────────┤
│                    过程对象 (Process)                 │
│  ProductionOrder/制造工单  Inspection/检验管理        │
├─────────────────────────────────────────────────────┤
│                    资源对象 (Resource)                │
│  Inventory/库存  WorkCenter/工作中心                  │
│  Equipment/设备  Asset/固定资产                      │
├─────────────────────────────────────────────────────┤
│                    财务对象 (Financial)               │
│  AR/应收账款  AP/应付账款  GL/总账  Cost/成本核算     │
├─────────────────────────────────────────────────────┤
│                    结构对象 (Structural)              │
│  BOM/物料清单                                        │
├─────────────────────────────────────────────────────┤
│                    事件对象 (Event)                   │
│  Shipment/发货                                       │
└─────────────────────────────────────────────────────┘
```

### 18个核心BusinessObject定义

#### BO-01：Material / 物料

| 字段 | 内容 |
|------|------|
| **对象编码** | `BO-MATERIAL` |
| **对象名称** | Material / 物料 |
| **对象类型** | 主数据对象 / 资源对象 |
| **业务域** | 库存管理 |
| **业务定义** | 制造企业的核心主数据，代表所有可以被采购、生产、存储、销售、消耗的物资。是ERP中关联模块最多（51张子表）、贯穿全价值链的基础参照对象 |
| **主标识** | 物料编码 (`item_code`) |
| **候选标识** | 物料GUID (`ITEM_ID`)、条码 (`barcode`)、规格+名称组合 |
| **生命周期状态** | 新建 → 审核 → 生效 → 停用 |
| **别名/同义词** | 品号、料号、Item、存货、零件、原料 |
| **跨系统对应** | ERP:ITEM / MES:materialNo / SRM:prodCode / TITA:Crm.Product |
| **关联对象** | Supplier(供应)、BOM(组成)、Inventory(库存)、PurchaseOrder(采购)、ProductionOrder(消耗/产出)、SalesOrder(销售)、Cost(成本) |
| **权限等级** | 内部 |
| **Agent可用性** | ✅ 可查询、分析、解释 |
